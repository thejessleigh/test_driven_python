Temperature

In this lesson you'll build two functions, f_to_c and c_to_f that will convert readings from one scale into another. For this particular example, we don't want to have to deal with decimal precision, so each function should return an integer reading after the conversion, and should round down.

Fahrenheit and Celsius

One of the key differences between Fahrenheit and Celsius is that the freezing point of water is 0 degrees on the Celsius scale, and 32 degrees on the Fahrenheit scale.

Beyond that, each degree of Fahrenheit is 5/9 of one degree Celsius.

How is a float different than an integer?

floats are a data type in Python that represent numbers. They're different from integers because floats can represent a larger subset of numbers. floats have decimal points. integers will always represent "whole" numbers.

Python3 uses float math when dividing. This might be a bit different than you're used to if you've worked with Python 2 before.

5 / 3 = 1.6666666666666667

For the purpose of this challenge, though, we want integers, so part of your challenge is to look up how to turn a float into an integer, and make sure it always rounds down.

Lesson objectives

  • Floating point math

Challenge Rules:

f_to_c
  • Accepts an integer or float
  • Converts a Fahrenheit reading to Celsius
  • Returns an integer
c_to_f
  • Accepts an integer or float
  • Converts a Celsius reading to Fahrenheit
  • Returns an integer

Example

freezing point
f_to_c(32) #=> 0
c_to_f(0) #=> 32
boiling point
f_to_c(212) #=> 100
c_to_f(100) #=> 212
