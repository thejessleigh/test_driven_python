Overview

Capitalizing the title of books and articles can be a bit tricky in English. There are a couple finnicky rules that mean you can't just capitalize every word and call it a day.

This lesson we're going to create a Class called Book. This class should take in a single parameter when initialize. This will be a string with the book's title.

You should then set the book's title attribute equal to the correctly "titleized" version of the input string.

Lesson Objectives

  1. Read official Python documentation
  2. Learn how to create a Class
  3. String manipulation
  4. Setting variables on an instance of a Class.

Challenge Rules

  • Create a class called Book that takes in a single string when initialized
  • Create a function inside your Book class that correctly "titleizes" the string according to the following rules:
    1. Capitalize the first and last word
    2. Capitalize nouns, pronouns, adjectives, verbs, adverbs, and subordinate conjunctions (although, because)
    3. Lowercase articles (a, an, the), coordinating conjunctions (and, but or), and prepositions
    4. Lowercase the 'to' in an infinitive (Let's Learn to Play Chess)
  • Assign the "titleized" string to an instance variable in Book called title.
